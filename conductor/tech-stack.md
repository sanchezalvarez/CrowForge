# Technology Stack

## 1. Programming Languages

*   **Python:** Used for the backend API and AI engine orchestration.
*   **TypeScript:** The primary language for the React frontend.
*   **Rust:** Used for the Tauri desktop application shell and backend integration.

## 2. Backend

*   **FastAPI:** A modern, high-performance web framework for building the backend API.

## 3. Frontend

*   **React:** A JavaScript library for building the user interface.
*   **Vite:** A fast build tool and development server for the frontend.

## 4. Desktop

*   **Tauri:** A framework for building lightweight, secure, and cross-platform desktop applications using web technologies.

## 5. Database

*   **SQLite:** A C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine.
